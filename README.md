# Path of Exile 2 Re-entry Protection & Area Timer

Приложение для мониторинга лог-файлов Path of Exile 2, которое помогает отслеживать время нахождения в зонах и предупреждает о необходимости перезахода.

## Особенности
- **Таймер перезахода**: Обратный отсчет при входе в отслеживаемую зону.
- **Цветовая индикация**: Плавный переход от зеленого (безопасно) к красному (пора перезаходить).
- **Общий таймер карты**: Считает время нахождения на текущей карте, сбрасывается только при новом Instance ID.
- **Звуковые уведомления**: Возможность выбрать свой WAV/MP3 файл.
- **Безопасность**: Приложение только читает текстовый файл `Client.txt`. Никакого чтения памяти или вмешательства в игру.

## Установка

1. Убедитесь, что у вас установлен Python 3.11+.
2. Клонируйте или скачайте проект.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск
```bash
python main.py
```

## Как пользоваться
1. Нажмите **"Select Game Folder"** и выберите корневую папку Path of Exile 2. Приложение автоматически найдет `logs\Client.txt`.
2. В списке **"Tracked Areas"** добавьте названия карт или зон, которые вы хотите отслеживать (например, `MapIceCave`).
3. Настройте **"Duration (s)"** для таймера перезахода.
4. (Опционально) Выберите свой звуковой файл и проверьте его кнопкой **"Test Sound"**.
5. Нажмите **"Start Monitoring"**.

## Валидация логики
Для проверки работы парсера и логики таймеров без запуска игры можно выполнить:
```bash
python validate_logic.py
```
