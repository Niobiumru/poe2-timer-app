# Path of Exile 2 Re-entry Protection & Area Timer

Приложение для мониторинга лог-файлов Path of Exile 2, которое помогает отслеживать время нахождения в зонах и предупреждает о необходимости перезахода.

## Особенности
- **Таймер перезахода**: Обратный отсчет при входе в отслеживаемую зону.
- **Цветовая индикация**: Плавный переход от зеленого (безопасно) к красному (пора перезаходить).
- **Общий таймер карты**: Считает время нахождения на текущей карте, сбрасывается только при новом Instance ID.
- **Звуковые уведомления**: Возможность выбрать свой WAV/MP3 файл.
- **Безопасность**: Приложение только читает текстовый файл `Client.txt`. Никакого чтения памяти или вмешательства в игру.

## 🚀 Быстрый старт (для пользователей)

Для использования программы вам не требуется устанавливать Python:
1. Перейдите на страницу [Релизов (Releases)](https://github.com/Niobiumru/poe2-timer-app/releases) и скачайте файл **`PoE2Timer.exe`** из последнего релиза.
2. Запустите скачанный файл.
3. В интерфейсе нажмите **"Select Game Folder"** и выберите корневую папку Path of Exile 2. Приложение автоматически найдет файл логов `logs\Client.txt`.
4. Нажмите **"Start Monitoring"** и наслаждайтесь игрой!

## 🛠️ Запуск из исходного кода (для разработчиков)

Если вы хотите запустить проект напрямую или изменить его код:
1. Убедитесь, что у вас установлен Python 3.11+.
2. Клонируйте репозиторий.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```

## Валидация логики (Тесты)
Для проверки работы парсера и логики таймеров без запуска игры выполните:
```bash
python tests/validate_logic.py
```
